96=head1 NAME
97
98Mojo::Loader - Load all kinds of things
99
100=head1 SYNOPSIS
101
102 use Mojo::Loader qw(data_section find_modules load_class);
103
104 # Find modules in a namespace
105 for my $module (find_modules 'Some::Namespace') {
106
107 # Load them safely
108 my $e = load_class $module;
109 warn qq{Loading "$module" failed: $e} and next if ref $e;
110
111 # And extract files from the DATA section
112 say data_section($module, 'some_file.txt');
113 }
114
115=head1 DESCRIPTION
116
117L<Mojo::Loader> is a class loader and plugin framework. Aside from finding
118modules and loading classes, it allows multiple files to be stored in the
119C<DATA> section of a class, which can then be accessed individually.
120
121 package Foo;
122
123 1;
124 __DATA__
125
126 @@ test.txt
127 This is the first file.
128
129 @@ test2.html (base64)
130 VGhpcyBpcyB0aGUgc2Vjb25kIGZpbGUu
131
132 @@ test
133 This is the
134 third file.
135
136Each file has a header starting with C<@@>, followed by the file name and
137optional instructions for decoding its content. Currently only the Base64
138encoding is supported, which can be quite convenient for the storage of binary
139data.
140
141=head1 FUNCTIONS
142
143L<Mojo::Loader> implements the following functions, which can be imported
144individually.
145
146=head2 data_section
147
148 my $all = data_section 'Foo::Bar';
149 my $index = data_section 'Foo::Bar', 'index.html';
150
151Extract embedded file from the C<DATA> section of a class, all files will be
152cached once they have been accessed for the first time.
153
154 # List embedded files
155 say for keys %{data_section 'Foo::Bar'};
156
157=head2 file_is_binary
158
159 my $bool = file_is_binary 'Foo::Bar', 'test.png';
160
161Check if embedded file from the C<DATA> section of a class was Base64 encoded.
162
163=head2 find_packages
164
165 my @pkgs = find_packages 'MyApp::Namespace';
166
167Search for packages in a namespace non-recursively.
168
169=head2 find_modules
170
171 my @modules = find_modules 'MyApp::Namespace';
172
173Search for modules in a namespace non-recursively.
174
175=head2 load_class
176
177 my $e = load_class 'Foo::Bar';
178
179Load a class and catch exceptions, returns a false value if loading was
180successful, a true value if the class has already been loaded, or a
181L<Mojo::Exception> object if loading failed. Note that classes are checked for a
182C<new> method to see if they are already loaded.
183
184 # Handle exceptions
185 if (my $e = load_class 'Foo::Bar') {
186 die ref $e ? "Exception: $e" : 'Not found!';
187 }
