127Mojo::IOLoop::Stream - Non-blocking I/O stream
128
129=head1 SYNOPSIS
130
131 use Mojo::IOLoop::Stream;
132
133 # Create stream
134 my $stream = Mojo::IOLoop::Stream->new($handle);
135 $stream->on(read => sub {

143 $stream->on(error => sub {

154
155=head1 DESCRIPTION
156
157L<Mojo::IOLoop::Stream> is a container for I/O streams used by L<Mojo::IOLoop>.
158
159=head1 EVENTS
160
161L<Mojo::IOLoop::Stream> inherits all events from L<Mojo::EventEmitter> and can
162emit the following new ones.
163
164=head2 close
165
166 $stream->on(close => sub {
167 my $stream = shift;
168 ...
169 });
170
171Emitted if the stream gets closed.
172
173=head2 drain
174
175 $stream->on(drain => sub {
176 my $stream = shift;
177 ...
178 });
179
180Emitted once all data has been written.
181
182=head2 error
183
184 $stream->on(error => sub {
185 my ($stream, $err) = @_;
186 ...
187 });
188
189Emitted if an error occurs on the stream, fatal if unhandled.
190
191=head2 read
192
193 $stream->on(read => sub {
194 my ($stream, $bytes) = @_;
195 ...
196 });
197
198Emitted if new data arrives on the stream.
199
200=head2 timeout
201
202 $stream->on(timeout => sub {
203 my $stream = shift;
204 ...
205 });
206
207Emitted if the stream has been inactive for too long and will get closed
208automatically.
209
210=head2 write
211
212 $stream->on(write => sub {
213 my ($stream, $bytes) = @_;
214 ...
215 });
216
217Emitted if new data has been written to the stream.
218
