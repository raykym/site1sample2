354=head1 NAME
355
356Mojo::DOM - Minimalistic HTML/XML DOM parser with CSS selectors
357
358=head1 SYNOPSIS
359
360 use Mojo::DOM;
361
362 # Parse
363 my $dom = Mojo::DOM->new('<div><p id="a">Test</p><p id="b">123</p></div>');
364
365 # Find
366 say $dom->at('#b')->text;
367 say $dom->find('p')->map('text')->join("\n");
368 say $dom->find('[id]')->map(attr => 'id')->join("\n");
369
370 # Iterate
371 $dom->find('p[id]')->reverse->each(sub { say $_->{id} });
372
373 # Loop
374 for my $e ($dom->find('p[id]')->each) {
375 say $e->{id}, ':', $e->text;
376 }
377
378 # Modify
379 $dom->find('div p')->last->append('<p id="c">456</p>');
380 $dom->find(':not(p)')->map('strip');
381
382 # Render
383 say "$dom";
384
385=head1 DESCRIPTION
386
387L<Mojo::DOM> is a minimalistic and relaxed HTML/XML DOM parser with CSS
388selector support. It will even try to interpret broken HTML and XML, so you
389should not use it for validation.
390
391=head1 NODES AND ELEMENTS
392
393When we parse an HTML/XML fragment, it gets turned into a tree of nodes.
394
395 <!DOCTYPE html>
396 <html>
397 <head><title>Hello</title></head>
398 <body>World!</body>
399 </html>
400
401There are currently eight different kinds of nodes, C<cdata>, C<comment>,
402C<doctype>, C<pi>, C<raw>, C<root>, C<tag> and C<text>. Elements are nodes of
403the type C<tag>.
404
405 root
406 |- doctype (html)
407 +- tag (html)
408 |- tag (head)
409 | +- tag (title)
410 | +- raw (Hello)
411 +- tag (body)
412 +- text (World!)
413
414While all node types are represented as L<Mojo::DOM> objects, some methods like
415L</"attr"> and L</"namespace"> only apply to elements.
416
417=head1 CASE-SENSITIVITY
418
419L<Mojo::DOM> defaults to HTML semantics, that means all tags and attribute
420names are lowercased and selectors need to be lowercase as well.
421
422 # HTML semantics
423 my $dom = Mojo::DOM->new('<P ID="greeting">Hi!</P>');
424 say $dom->at('p[id]')->text;
425
426If an XML declaration is found, the parser will automatically switch into XML
427mode and everything becomes case-sensitive.
428
429 # XML semantics
430 my $dom = Mojo::DOM->new('<?xml version="1.0"?><P ID="greeting">Hi!</P>');
431 say $dom->at('P[ID]')->text;
432
433XML detection can also be disabled with the L</"xml"> method.
434
435 # Force XML semantics
436 my $dom = Mojo::DOM->new->xml(1)->parse('<P ID="greeting">Hi!</P>');
437 say $dom->at('P[ID]')->text;
438
439 # Force HTML semantics
440 my $dom = Mojo::DOM->new->xml(0)->parse('<P ID="greeting">Hi!</P>');
441 say $dom->at('p[id]')->text;
442
443=head1 METHODS
444
445L<Mojo::DOM> implements the following methods.
446
447=head2 all_text
448
449 my $text = $dom->all_text;
450
451Extract text content from all descendant nodes of this element.
452
453 # "foo\nbarbaz\n"
454 $dom->parse("<div>foo\n<p>bar</p>baz\n</div>")->at('div')->all_text;
455
456=head2 ancestors
457
458 my $collection = $dom->ancestors;
459 my $collection = $dom->ancestors('div ~ p');
460
461Find all ancestor elements of this node matching the CSS selector and return a
462L<Mojo::Collection> object containing these elements as L<Mojo::DOM> objects.
463All selectors from L<Mojo::DOM::CSS/"SELECTORS"> are supported.
464
465 # List tag names of ancestor elements
466 say $dom->ancestors->map('tag')->join("\n");
467
468=head2 append
469
470 $dom = $dom->append('<p>I ♥ Mojolicious!</p>');
471
472Append HTML/XML fragment to this node (for all node types other than C<root>).
473
474 # "<div><h1>Test</h1><h2>123</h2></div>"
475 $dom->parse('<div><h1>Test</h1></div>')
476 ->at('h1')->append('<h2>123</h2>')->root;
477
478 # "<p>Test 123</p>"
479 $dom->parse('<p>Test</p>')->at('p')
480 ->child_nodes->first->append(' 123')->root;
481
482=head2 append_content
483
484 $dom = $dom->append_content('<p>I ♥ Mojolicious!</p>');
485
486Append HTML/XML fragment (for C<root> and C<tag> nodes) or raw content to this
487node's content.
488
489 # "<div><h1>Test123</h1></div>"
490 $dom->parse('<div><h1>Test</h1></div>')
491 ->at('h1')->append_content('123')->root;
492
493 # "<!-- Test 123 --><br>"
494 $dom->parse('<!-- Test --><br>')
495 ->child_nodes->first->append_content('123 ')->root;
496
497 # "<p>Test<i>123</i></p>"
498 $dom->parse('<p>Test</p>')->at('p')->append_content('<i>123</i>')->root;
499
500=head2 at
501
502 my $result = $dom->at('div ~ p');
503
504Find first descendant element of this element matching the CSS selector and
505return it as a L<Mojo::DOM> object, or C<undef> if none could be found. All
506selectors from L<Mojo::DOM::CSS/"SELECTORS"> are supported.
507
508 # Find first element with "svg" namespace definition
509 my $namespace = $dom->at('[xmlns\:svg]')->{'xmlns:svg'};
510
511=head2 attr
512
513 my $hash = $dom->attr;
514 my $foo = $dom->attr('foo');
515 $dom = $dom->attr({foo => 'bar'});
