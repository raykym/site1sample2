36=head1 SYNOPSIS
37
38 package MyApp;
39 use Mojo::Base 'Mojo';

41 # All the complexities of CGI, PSGI, HTTP and WebSockets get reduced to a
42 # single method call!
43 sub handler {
44 my ($self, $tx) = @_;
45
46 # Request
47 my $method = $tx->req->method;
48 my $path = $tx->req->url->path;
49
50 # Response
51 $tx->res->code(200);

59=head1 DESCRIPTION
60
61A powerful web development toolkit, with all the basic tools and helpers needed
62to write simple web applications and higher level web frameworks, such as
63L<Mojolicious>. Some of the most commonly used tools are L<Mojo::UserAgent>,
64L<Mojo::DOM>, L<Mojo::JSON>, L<Mojo::Server::Daemon>, L<Mojo::Server::Prefork>,
65L<Mojo::IOLoop> and L<Mojo::Template>.
66
67See L<Mojolicious::Guides> for more!
68
69=head1 ATTRIBUTES
70
71L<Mojo> implements the following attributes.
72
73=head2 home
74
75 my $home = $app->home;
76 $app = $app->home(Mojo::Home->new);
77
78The home directory of your application, defaults to a L<Mojo::Home> object
79which stringifies to the actual path.
80
81 # Generate portable path relative to home directory
82 my $path = $app->home->rel_file('data/important.txt');
83
84=head2 log
85
86 my $log = $app->log;
87 $app = $app->log(Mojo::Log->new);
88
89The logging layer of your application, defaults to a L<Mojo::Log> object.
90
91 # Log debug message
92 $app->log->debug('It works');
93
94=head2 ua
95
96 my $ua = $app->ua;
97 $app = $app->ua(Mojo::UserAgent->new);
98
99A full featured HTTP user agent for use in your applications, defaults to a
100L<Mojo::UserAgent> object.
101
102 # Perform blocking request
103 say $app->ua->get('example.com')->res->body;
104
105=head1 METHODS
106
107L<Mojo> inherits all methods from L<Mojo::Base> and implements the following
108new ones.
109
110=head2 build_tx
111
112 my $tx = $app->build_tx;
113
114Transaction builder, defaults to building a L<Mojo::Transaction::HTTP> object.
115
116=head2 config
117
118 my $hash = $app->config;
119 my $foo = $app->config('foo');
120 $app = $app->config({foo => 'bar', baz => 23});

122
123Application configuration.
124
125 # Remove value
126 my $foo = delete $app->config->{foo};
127
128 # Assign multiple values at once
129 $app->config(foo => 'test', bar => 23);
130
131=head2 handler
132

136be called for each new transaction, which will usually be a
137L<Mojo::Transaction::HTTP> or L<Mojo::Transaction::WebSocket> object. Meant to
138be overloaded in a subclass.
