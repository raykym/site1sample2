297=head2 any
298
299 my $route = $r->any;
300 my $route = $r->any('/:foo');
301 my $route = $r->any('/:foo' => sub {...});
302 my $route = $r->any('/:foo' => sub {...} => 'name');
303 my $route = $r->any('/:foo' => {foo => 'bar'} => sub {...});
304 my $route = $r->any('/:foo' => [foo => qr/\w+/] => sub {...});
305 my $route = $r->any('/:foo' => (agent => qr/Firefox/) => sub {...});
306 my $route = $r->any(['GET', 'POST'] => '/:foo' => sub {...});
307 my $route = $r->any(['GET', 'POST'] => '/:foo' => [foo => qr/\w+/]);
308
309Generate L<Mojolicious::Routes::Route> object matching any of the listed HTTP
310request methods or all.
311
312 # Route with pattern and destination
313 $r->any('/user')->to('user#whatever');
314
315All arguments are optional, but some have to appear in a certain order, like the
316two supported array reference values, which contain the HTTP methods to match
317and restrictive placeholders.
318
319 # Route with HTTP methods, pattern, restrictive placeholders and destination
320 $r->any(['DELETE', 'PUT'] => '/:foo' => [foo => qr/\w+/])->to('foo#bar');
321
322There are also two supported string values, containing the route pattern and the
323route name, defaulting to the pattern C</> and a name based on the pattern.
324
325 # Route with pattern, name and destination
326 $r->any('/:foo' => 'foo_route')->to('foo#bar');
327
328An arbitrary number of key/value pairs in between the route pattern and name can
329be used to specify route conditions.
330
331 # Route with pattern, condition and destination
332 $r->any('/' => (agent => qr/Firefox/))->to('foo#bar');
333
334A hash reference is used to specify optional placeholders and default values for
335the stash.
336
337 # Route with pattern, optional placeholder and destination
338 $r->any('/:foo' => {foo => 'bar'})->to('foo#bar');
339
340And a code reference can be used to specify a C<cb> value to be merged into the
341default values for the stash.
342
343 # Route with pattern and a closure as destination
344 $r->any('/:foo' => sub {
345 my $c = shift;
346 $c->render(text => 'Hello World!');
347 });
348
349See L<Mojolicious::Guides::Tutorial> and L<Mojolicious::Guides::Routing> for
350more information.
