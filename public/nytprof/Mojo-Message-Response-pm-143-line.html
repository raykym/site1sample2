152=head1 NAME
153
154Mojo::Message::Response - HTTP response
155
156=head1 SYNOPSIS
157
158 use Mojo::Message::Response;
159
160 # Parse
161 my $res = Mojo::Message::Response->new;
162 $res->parse("HTTP/1.0 200 OK\x0d\x0a");
163 $res->parse("Content-Length: 12\x0d\x0a");
164 $res->parse("Content-Type: text/plain\x0d\x0a\x0d\x0a");
165 $res->parse('Hello World!');
166 say $res->code;
167 say $res->headers->content_type;
168 say $res->body;
169
170 # Build
171 my $res = Mojo::Message::Response->new;
172 $res->code(200);
173 $res->headers->content_type('text/plain');
174 $res->body('Hello World!');
175 say $res->to_string;
176
177=head1 DESCRIPTION
178
179L<Mojo::Message::Response> is a container for HTTP responses, based on
180L<RFC 7230|http://tools.ietf.org/html/rfc7230> and
181L<RFC 7231|http://tools.ietf.org/html/rfc7231>.
182
183=head1 EVENTS
184
185L<Mojo::Message::Response> inherits all events from L<Mojo::Message>.
186
187=head1 ATTRIBUTES
188
189L<Mojo::Message::Response> inherits all attributes from L<Mojo::Message> and
190implements the following new ones.
191
192=head2 code
193
194 my $code = $res->code;
195 $res = $res->code(200);
196
197HTTP response status code.
198
199=head2 message
200
201 my $msg = $res->message;
202 $res = $res->message('OK');
203
204HTTP response status message.
205
206=head1 METHODS
207
208L<Mojo::Message::Response> inherits all methods from L<Mojo::Message> and
209implements the following new ones.
