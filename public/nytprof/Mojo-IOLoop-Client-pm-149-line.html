201=head1 NAME
202
203Mojo::IOLoop::Client - Non-blocking TCP client
204
205=head1 SYNOPSIS
206
207 use Mojo::IOLoop::Client;
208
209 # Create socket connection
210 my $client = Mojo::IOLoop::Client->new;
211 $client->on(connect => sub {

222 $client->reactor->start unless $client->reactor->is_running;
223
224=head1 DESCRIPTION
225
226L<Mojo::IOLoop::Client> opens TCP connections for L<Mojo::IOLoop>.
227
228=head1 EVENTS
229
230L<Mojo::IOLoop::Client> inherits all events from L<Mojo::EventEmitter> and can
231emit the following new ones.
232
233=head2 connect
234
235 $client->on(connect => sub {
236 my ($client, $handle) = @_;
237 ...
238 });
239
240Emitted once the connection is established.
241
242=head2 error
243
244 $client->on(error => sub {
245 my ($client, $err) = @_;
246 ...
247 });
248
249Emitted if an error occurs on the connection, fatal if unhandled.
250
251=head1 ATTRIBUTES
252
253L<Mojo::IOLoop::Client> implements the following attributes.
254
255=head2 reactor
256
257 my $reactor = $client->reactor;
