264=head1 DESCRIPTION
265

267L<RFC 7230|http://tools.ietf.org/html/rfc7230>,
268L<RFC 7231|http://tools.ietf.org/html/rfc7231>,
269L<RFC 7235|http://tools.ietf.org/html/rfc7235> and
270L<RFC 2817|http://tools.ietf.org/html/rfc2817>.
271

273
