248=head2 classes

253Classes to use for finding templates in C<DATA> sections with L<Mojo::Loader>,
254first one has the highest precedence, defaults to C<main>. Only files with
255exactly two extensions will be used, like C<index.html.ep>. Note that for
256templates to be detected, these classes need to have already been loaded and
257added before L</"warmup"> is called, which usually happens automatically during
258application startup.

323=head2 accepts
324
325 my $all = $renderer->accepts(Mojolicious::Controller->new);
326 my $best = $renderer->accepts(Mojolicious::Controller->new, 'html', 'json');
327
328Select best possible representation for L<Mojolicious::Controller> object from
329C<Accept> request header, C<format> stash value or C<format> C<GET>/C<POST>
330parameter, defaults to returning the first extension if no preference could be
331detected. Since browsers often don't really know what they actually want,
332unspecific C<Accept> request headers with more than one MIME type will be
333ignored, unless the C<X-Requested-With> header is set to the value
334C<XMLHttpRequest>.
