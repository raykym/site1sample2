200=head1 NAME
201
202Mojolicious::Routes - Always find your destination with routes
203
204=head1 SYNOPSIS
205
206 use Mojolicious::Routes;
207
208 # Simple route
209 my $r = Mojolicious::Routes->new;
210 $r->route('/')->to(controller => 'blog', action => 'welcome');
211
212 # More advanced routes
213 my $blog = $r->under('/blog');
214 $blog->get('/list')->to('blog#list');
215 $blog->get('/:id' => [id => qr/\d+/])->to('blog#show', id => 23);
216 $blog->patch(sub { shift->render(text => 'Go away!', status => 405) });
217
218=head1 DESCRIPTION
219
220L<Mojolicious::Routes> is the core of the L<Mojolicious> web framework.
221
222See L<Mojolicious::Guides::Routing> for more.
223
224=head1 ATTRIBUTES
225
226L<Mojolicious::Routes> inherits all attributes from
227L<Mojolicious::Routes::Route> and implements the following new ones.
228
229=head2 base_classes
230
231 my $classes = $r->base_classes;
232 $r = $r->base_classes(['MyApp::Controller']);
233
234Base classes used to identify controllers, defaults to
235L<Mojolicious::Controller> and L<Mojo>.
236
237=head2 cache
238
239 my $cache = $r->cache;
