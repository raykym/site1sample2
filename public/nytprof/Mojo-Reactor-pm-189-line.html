34Mojo::Reactor - Low-level event reactor base class
35

54=head1 DESCRIPTION
55
56L<Mojo::Reactor> is an abstract base class for low-level event reactors, like
57L<Mojo::Reactor::EV> and L<Mojo::Reactor::Poll>.
58
59=head1 EVENTS
60
61L<Mojo::Reactor> inherits all events from L<Mojo::EventEmitter> and can emit
62the following new ones.
63
64=head2 error
65
66 $reactor->on(error => sub {
67 my ($reactor, $err) = @_;
68 ...
69 });
70
71Emitted for exceptions caught in callbacks, fatal if unhandled. Note that if
72this event is unhandled or fails it might kill your program, so you need to be
73careful.
74
75 $reactor->on(error => sub {
76 my ($reactor, $err) = @_;
77 say "Something very bad happened: $err";
78 });
79
80=head1 METHODS
81
82L<Mojo::Reactor> inherits all methods from L<Mojo::EventEmitter> and implements
83the following new ones.
84
85=head2 again
86
87 $reactor->again($id);
88
89Restart timer. Meant to be overloaded in a subclass. Note that this method
90requires an active timer.
91
92=head2 detect
93
94 my $class = Mojo::Reactor->detect;
95
96Detect and load the best reactor implementation available, will try the value
97of the C<MOJO_REACTOR> environment variable, L<Mojo::Reactor::EV> or
98L<Mojo::Reactor::Poll>.
99
100 # Instantiate best reactor implementation available
101 my $reactor = Mojo::Reactor->detect->new;
102
103=head2 io
104
105 $reactor = $reactor->io($handle => sub {...});
106
107Watch handle for I/O events, invoking the callback whenever handle becomes
108readable or writable. Meant to be overloaded in a subclass.
109
110 # Callback will be executed twice if handle becomes readable and writable
111 $reactor->io($handle => sub {
112 my ($reactor, $writable) = @_;
113 say $writable ? 'Handle is writable' : 'Handle is readable';
114 });
115
116=head2 is_running
117
118 my $bool = $reactor->is_running;
119
120Check if reactor is running. Meant to be overloaded in a subclass.
121
122=head2 next_tick
123
124 my $undef = $reactor->next_tick(sub {...});
125
126Execute callback as soon as possible, but not before returning or other
127callbacks that have been registered with this method, always returns C<undef>.
128Meant to be overloaded in a subclass.
129
130=head2 one_tick
131
132 $reactor->one_tick;
133
134Run reactor until an event occurs. Note that this method can recurse back into
135the reactor, so you need to be careful. Meant to be overloaded in a subclass.
136
137 # Don't block longer than 0.5 seconds
138 my $id = $reactor->timer(0.5 => sub {});
139 $reactor->one_tick;
140 $reactor->remove($id);
141
142=head2 recurring
143
144 my $id = $reactor->recurring(0.25 => sub {...});
145
146Create a new recurring timer, invoking the callback repeatedly after a given
147amount of time in seconds. Meant to be overloaded in a subclass.
148
149=head2 remove
150
151 my $bool = $reactor->remove($handle);
152 my $bool = $reactor->remove($id);
153
154Remove handle or timer. Meant to be overloaded in a subclass.
155
156=head2 reset
157
158 $reactor->reset;
159
160Remove all handles and timers. Meant to be overloaded in a subclass.
161
162=head2 start
163
164 $reactor->start;
165
166Start watching for I/O and timer events, this will block until L</"stop"> is
167called. Note that some reactors stop automatically if there are no events being
168watched anymore. Meant to be overloaded in a subclass.
169
170 # Start reactor only if it is not running already
171 $reactor->start unless $reactor->is_running;
172
173=head2 stop
174
175 $reactor->stop;
176
177Stop watching for I/O and timer events. Meant to be overloaded in a subclass.
178
179=head2 timer
180
181 my $id = $reactor->timer(0.5 => sub {...});
182
183Create a new timer, invoking the callback after a given amount of time in
184seconds. Meant to be overloaded in a subclass.
185
186=head2 watch
187
188 $reactor = $reactor->watch($handle, $readable, $writable);
189
190Change I/O events to watch handle for with true and false values. Meant to be
191overloaded in a subclass. Note that this method requires an active I/O watcher.
192
193 # Watch only for readable events
194 $reactor->watch($handle, 1, 0);
195
196 # Watch only for writable events
197 $reactor->watch($handle, 0, 1);
198
199 # Watch for readable and writable events
200 $reactor->watch($handle, 1, 1);
201
202 # Pause watching for events
203 $reactor->watch($handle, 0, 0);
