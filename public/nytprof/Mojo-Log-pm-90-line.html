94=head1 DESCRIPTION
95
96L<Mojo::Log> is a simple logger for L<Mojo> projects.
97
98=head1 EVENTS
99
100L<Mojo::Log> inherits all events from L<Mojo::EventEmitter> and can emit the
101following new ones.
102
103=head2 message
104
105 $log->on(message => sub {
106 my ($log, $level, @lines) = @_;
107 ...
108 });
109
110Emitted when a new message gets logged.
111
112 $log->unsubscribe('message')->on(message => sub {
113 my ($log, $level, @lines) = @_;
114 say "$level: ", @lines;
115 });
116
