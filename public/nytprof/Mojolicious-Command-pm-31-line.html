118=head1 DESCRIPTION
119
120L<Mojolicious::Command> is an abstract base class for L<Mojolicious> commands.
121
122See L<Mojolicious::Commands/"COMMANDS"> for a list of commands that are
123available by default.
124
125=head1 ATTRIBUTES
126
127L<Mojolicious::Command> implements the following attributes.
128
129=head2 app
130
131 my $app = $command->app;
132 $command = $command->app(Mojolicious->new);
133
134Application for command, defaults to a L<Mojo::HelloWorld> object.
135
136 # Introspect
137 say "Template path: $_" for @{$command->app->renderer->paths};
138
139=head2 description
140
141 my $description = $command->description;
142 $command = $command->description('Foo');
143
144Short description of command, used for the command list.
145
146=head2 quiet
147
148 my $bool = $command->quiet;
149 $command = $command->quiet($bool);
150
151Limited command output.
152
153=head2 usage
154
155 my $usage = $command->usage;
156 $command = $command->usage('Foo');
157
158Usage information for command, used for the help screen.
159
