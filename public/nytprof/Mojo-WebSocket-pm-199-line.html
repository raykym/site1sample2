172=head1 NAME

174Mojo::WebSocket - The WebSocket protocol
175
176=head1 SYNOPSIS
177
178 use Mojo::WebSocket qw(WS_TEXT build_frame parse_frame);
179
180 my $bytes = build_frame 0, 1, 0, 0, 0, WS_TEXT, 'Hello World!';

182
183=head1 DESCRIPTION
184
185L<Mojo::WebSocket> implements the WebSocket protocol as described in
186L<RFC 6455|http://tools.ietf.org/html/rfc6455>. Note that 64-bit frames require
187a Perl with support for quads or they are limited to 32-bit.
188
189=head1 FUNCTIONS
190
191L<Mojo::WebSocket> implements the following functions, which can be imported
192individually.
193
194=head2 build_frame
195
196 my $bytes = build_frame $masked, $fin, $rsv1, $rsv2, $rsv3, $op, $payload;
197
198Build WebSocket frame.
199
200 # Masked binary frame with FIN bit and payload
201 say build_frame 1, 1, 0, 0, 0, WS_BINARY, 'Hello World!';
202
203 # Text frame with payload but without FIN bit
204 say build_frame 0, 0, 0, 0, 0, WS_TEXT, 'Hello ';
205
206 # Continuation frame with FIN bit and payload
207 say build_frame 0, 1, 0, 0, 0, WS_CONTINUATION, 'World!';
208
209 # Close frame with FIN bit and without payload
210 say build_frame 0, 1, 0, 0, 0, WS_CLOSE, '';
211
212 # Ping frame with FIN bit and payload
213 say build_frame 0, 1, 0, 0, 0, WS_PING, 'Test 123';
214
215 # Pong frame with FIN bit and payload
216 say build_frame 0, 1, 0, 0, 0, WS_PONG, 'Test 123';
